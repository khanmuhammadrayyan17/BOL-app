# BOL — Panda Pet App 🐼

A fun, interactive virtual pet app built for children to practice daily routines, speech, and learning activities through caring for their own panda pet. Built as an HCI Design project using React, TypeScript, and Vite.

## Getting Started (Run Locally)

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Setup

```bash
# 1. Clone the repository
git clone <repo-url>
cd BOL

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

The app will be running at `http://localhost:5173` (default Vite port).

### Other Commands

```bash
# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## What Is This Project?

BOL is a virtual panda pet companion app designed for kids. The idea is that children interact with their panda through everyday care activities — each activity doubles as a fun learning or speech exercise.

### Key Features

- **Daily Care Activities** — Brush, Feed, Bath, Sleep, Play, and Walk your panda
- **Speech & Language** — Morning Greeting, Interview Mode, and Listen & Learn exercises to practice speaking
- **Learning & Creativity** — Story Time, Pet Diary, Pet Garden, and Tricks to build vocabulary and creativity
- **Challenge Mode** — Gamified challenges with rankings and friends
- **Pet Doctor / Vet Visit** — Learn about health through caring for your pet
- **Profile & Social** — User profiles, rankings, and the ability to add friends
- **Diagnostic Flow** — Onboarding assessment to tailor content to the child's level

### Tech Stack

- **React 18** with **TypeScript**
- **Vite** for fast dev/build
- **Tailwind CSS v4** for styling
- **React Router v7** for navigation
- **Radix UI** for accessible UI primitives
- **Motion** (Framer Motion) for animations
- **Recharts** for data visualization
- **Lucide React** for icons
